# Grid 栅格
<br/>

## 基本使用


grid 是由Row与Col组成的12栅格布局。

xs:<576px
sm:≥576px
md:≥768px
lg:≥992px
xl:≥1200px
xxl:≥1600px

Row是一行，col是行内元素。简单情况就小于576和大于576就可以了。一行相同种类数字之和等于12。
