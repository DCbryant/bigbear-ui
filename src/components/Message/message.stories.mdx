## message 函数调用

第一个参数文本，第二个参数是图标，可以不传或者undefined来使用默认图标，第三个参数是方位。

<Preview>
  <Story name='message3'>
      <div>
        <Button onClick={()=>message.success('成功的消息')}> success</Button>
        <Button onClick={()=>message.warning('警告的消息')}> warning</Button>
        <Button onClick={()=>message.danger('危险的消息')}> danger</Button>
        <Button onClick={()=>message.info('普通通知')}> info</Button>
        <Button onClick={()=>message.default('普通消息')}> default</Button>
        <Button onClick={()=>message.primary('主题通知',undefined,'rb')}> primary</Button>
        <Button onClick={()=>message.secondary('副标题通知',undefined,'lt')}> secondary</Button>
        <Button onClick={()=>message.light('浅色通知',<Icon icon='microphone' theme='dark'></Icon>)}> light</Button>
        <Button onClick={()=>message.dark('深色通知')}> dark</Button>
        <div className="zoom-in-topmesssage-enter"></div>
      </div>
  </Story>
</Preview>
