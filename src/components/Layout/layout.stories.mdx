如果要使用横向布局，比如sider和content是一个横向,那么给包裹的Layout传递row这个参数。


sider默认宽200，content默认最小宽度300,写了个媒体查询，在宽度小于520下sider单独一行（20是滚动条宽度）。

doc上展示由于preview有边框所以并不是很准确。
