# Carousel 轮播图
<br/>

## 基本使用

由Carousel组件包住需要轮播的节点即可使用。

由于是简化版的，功能有限，需要更多定制可以去安装react-slick。

高度需要自行传入，默认高度200px，传入容器也要设定高度，宽度自适应。
