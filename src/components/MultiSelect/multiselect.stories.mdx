# MultiSelect 多选框

<br/>

## 基本使用

多选框是在alert组件上进行的封装。

多选框封装不同于别的组件库，只要传数据就可以了，避免使用上过于复杂。

<Preview>
    <Story name='multiselect'>
        <div style={{height:'300px'}}>
        <MultiSelect  data={['options1','options2','options3','options4']} ></MultiSelect>
         </div>
    </Story>
</Preview>
